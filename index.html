La structure d'une page HTML

le doctype : première ligne qui definit que le fichier est de type document HTML.
le head : Partie déclarative du fichier, elle comprend les balises Title, meta, scripts additionnels. N'est pas affichée.
le body : contient le code qui génère la page dans le navigateur

les balises structurantes:

  • <p></p>: délimite un paragraphe,
  • <br/> : c'est une balise orpheline qui ajoute un retour à la ligne,
  • <hx></hx>: balises titres, x variant de 1 à 6 par ordre d'importance.
    h1 pour un titre important, h2 pour un sous-titre, etc..
  • <em></em>: mise en valeur faible d'un mot, d'une phrase,
  • <strong></strong>: mise en valeur plus importante,

Les listes:

  • <ul></ul>: liste à puce non ordonnée,
  • <ol></ol>: liste ordonnée (numérotée),
  • <li></li>: délimite un élément de liste ul et ol,

Les liens:

  • <a href="http://lien_externe.html"></a>: lien absolu qui référence un lien externe à la page,
  • <a href="[chemin/]ma_page_interne.html"></a>: lien relatif qui référence un lien interne,
  • <a href="#"></a>: lien vide,
  • <a href="#Ancre"></a>: lien qui référence une ancre de la page en cours,
  • <a href="[chemin/]ma_page_interne.html#Ancre"></a>: lien relatif vers une ancre d'un lien interne,

  • <a href="mailto:"monAdresse@mail"></a>: lien vers une adresse mail,
  • <a href="fic_download.zip"></a>: lien de téléchargement du fichier fic_download.zip,
Attributs des liens:
  • title : affiche une info-bulle lors du pointage de la souris,
  • target="_blanck" : ouvre le lien dans une nouvelle fenetre,

Les images: Elements essentiels d'un site, trois formats principaux suivant l'usage:
  • JPEG : format léger adapté aux images,
  • PNG : pour les graphiques hors images, codage en 8 et 24 bits qui n'altère pas la qualité d'image et permet les effets de transparence,
  • GIF : pour les images animées,

Une image est insérée via la balise orpheline <img>
l'attribut src="chemin" indique l'adresse de l'image
alt ="description" : texte alternatif et descriptif de l'image. Apparait lorsque l'image ne peut pas etre affichée,
title="message" : affichage infobulle.
Pour une image "lourde", on peut afficher une miniature (thumbnails) et ajouter un lien vers l'image en taille réelle.
exemple:
<a href="web/img/simplon_gateaux.jpg"> <img src="web/img/htmlCss.png">

Les figures:
Les figures sont des éléments qui viennent agrémenter un texte.
Une figure peut prendre la forme d'une image, d'un code source, d'une citation, etc..
